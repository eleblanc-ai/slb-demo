1. Extract the *.zip file to a folder on your computer.

2. Open the Command Prompt, Windows PowerShell (Windows), or Terminal (Mac).

3. Change the directory to the folder where you extracted the files: Type "cd <file/path>" and press Enter/Return.
e.g. "cd C:\Users\yourname\Documents\smart-lesson-plan-builder"

4. Type "node app.js" and press Enter/Return.

5. Copy the link after "Server running on" (http://localhost:3000), paste it into your browser, and press Enter/Return.

**Keep Command Prompt/PowerShell/Terminal running to continue running the program.
